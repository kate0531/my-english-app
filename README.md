# 영어 학습 채팅 앱

Next.js와 Tailwind CSS로 만든 영어 문법 교정 채팅형 학습 앱입니다.

## 기능

- 💬 채팅형 대화 인터페이스
- ✏️ 실시간 문법 교정
- 🎤 마이크 버튼으로 샘플 문장 입력
- 👆 사용자 말풍선 클릭으로 교정 문장 토글 표시
- 🎨 심플하고 모던한 디자인 (흰색 메인 + 마젠타 강조)

## 🚀 빠른 시작 (초보자용)

### 1단계: 터미널 열기
Cursor 에디터 하단의 터미널을 엽니다 (또는 `Ctrl + ` ` 키)

### 2단계: 패키지 설치
터미널에 아래 명령어를 입력하고 Enter:
```bash
npm install
```
(처음에는 몇 분 걸릴 수 있습니다)

### 3단계: 개발 서버 실행
설치가 완료되면 다음 명령어 입력:
```bash
npm run dev
```

### 4단계: 브라우저에서 확인
터미널에 `ready - started server on 0.0.0.0:3000` 메시지가 보이면,
웹 브라우저(Chrome, Edge 등)를 열고 주소창에 입력:
```
http://localhost:3000
```

✅ 이제 영어 학습 앱을 사용할 수 있습니다!

> 💡 **팁**: 서버를 중지하려면 터미널에서 `Ctrl + C`를 누르세요.

## 사용 방법

1. 텍스트 입력창에 영어 문장을 입력하거나
2. 마이크 버튼을 클릭하여 샘플 문장을 입력한 후
3. 전송 버튼을 클릭하면 문법 교정이 진행됩니다
4. 교정이 필요한 문장의 경우, 사용자 말풍선을 클릭하면 교정된 문장을 확인할 수 있습니다

## LLM API 연동

현재는 샘플 교정 로직이 구현되어 있습니다. 실제 LLM API를 사용하려면 `app/api/correct/route.ts` 파일의 주석 처리된 부분을 활성화하고 OpenAI API 키를 설정하세요.
