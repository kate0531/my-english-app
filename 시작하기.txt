==========================================
영어 학습 앱 실행 방법 (초보자용)
==========================================

이 파일은 개발자가 아닌 분들을 위한 간단한 실행 가이드입니다.

[방법 1: Cursor 터미널 사용]

1. Cursor 에디터 하단에 있는 터미널 창을 엽니다
   (또는 Ctrl + ` 키를 누릅니다)

2. 터미널에 아래 명령어를 하나씩 입력하고 Enter를 누릅니다:

   npm install
   
   (이 명령어는 패키지를 설치합니다. 몇 분 걸릴 수 있습니다)

3. 설치가 완료되면 다음 명령어를 입력합니다:

   npm run dev

4. 터미널에 다음과 같은 메시지가 나타납니다:
   "ready - started server on 0.0.0.0:3000, url: http://localhost:3000"

5. 웹 브라우저(Chrome, Edge 등)를 열고 주소창에 입력:
   http://localhost:3000

6. 영어 학습 채팅 화면이 나타납니다!


[방법 2: 명령 프롬프트(CMD) 사용]

1. Windows 키를 누르고 "cmd"를 입력한 후 Enter

2. 다음 명령어로 프로젝트 폴더로 이동:
   cd "c:\Users\이윤지\.cursor\test"

3. npm install 입력 후 Enter

4. npm run dev 입력 후 Enter

5. 브라우저에서 http://localhost:3000 열기


[중요 사항]

- Node.js가 설치되어 있어야 합니다
- 개발 서버를 중지하려면 터미널에서 Ctrl + C를 누르세요
- 브라우저를 닫아도 서버는 계속 실행됩니다
- 서버를 중지하면 브라우저에서 페이지를 볼 수 없습니다


[문제 해결]

Q: "npm이 인식되지 않습니다" 오류가 나요
A: Node.js를 설치해야 합니다. https://nodejs.org 에서 다운로드하세요

Q: 포트가 이미 사용 중이라는 오류가 나요
A: 다른 프로그램이 3000번 포트를 사용 중입니다. 
   터미널에서 Ctrl + C로 서버를 중지하고 다시 시도하세요

Q: 브라우저에서 "연결할 수 없습니다"가 나와요
A: npm run dev 명령어가 실행 중인지 확인하세요.

   git add .

   git commit -m "first commit"

   git branch -M main

   git remote add origin https://github.com/kate0531/selena.git

   git push -u origin main

   # 빌드 파일을 생성하려면 아래 명령어를 실행하세요:
   npm run build

   # Git 저장소를 초기화하려면 아래 명령어를 입력하세요:
